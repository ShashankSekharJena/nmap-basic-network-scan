# Basic Network Scanning with Nmap

## Objective
To perform a basic network scan using Nmap to identify open ports and services.

## Tool Used
- Nmap

## Target
- Local VM IP: 10.0.2.15

## Scan Result
- Host is reachable
- No open TCP ports found
- All scanned ports are closed

## Conclusion
The system does not expose any active network services, indicating a secure default configuration.
